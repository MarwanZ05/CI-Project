# Clustering-Based Customer Segmentation with CI/EC Algorithms

Building a comprehensive, submission-ready academic project to benchmark Computational Intelligence (CI) and Evolutionary Computation (EC) algorithms against K-Means for customer segmentation.

## Setup

```bash
pip install -r requirements.txt
```

## Running the Application

To explore the algorithms visually and tweak parameters:

```bash
streamlit run app.py
```

## Running Systematic Experiments

Run the following command to systematically benchmark and store 30 runs per algorithm configuration:

```bash
python experiments.py
```
